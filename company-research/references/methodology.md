# 一级市场投研方法论

## 投资研究框架

### 核心原则

1. **投资人思维**：发现矛盾、验证假设、评估风险，不只是总结
2. **证据导向**：所有判断基于具体证据，区分强/中/弱证据
3. **不编造**：资料中没有的信息标记"资料未提供"
4. **盲区优先**：识别资料未涉及但投决必知的信息

### 证据强度分级

| 等级 | 含义 | 典型来源 |
|------|------|---------|
| 强证据 | 可直接作为判断依据 | 财务数据、合同、订单、多份资料交叉验证 |
| 中证据 | 有一定参考价值但需验证 | 管理层访谈、BP、业务说明、部分数据支持 |
| 弱证据 | 仅作线索需进一步核实 | 单方面表述、口头承诺、未经验证的预测 |

## 投资盲区识别方法

### 什么是投资盲区？

资料中没有涉及，但做出投资决策必须知道的信息。比"资料缺失"更深一层——不是字段为空，而是投决逻辑链上的关键缺口。

### 六类盲区

| 类型 | 识别逻辑 | 示例 |
|------|---------|------|
| 产品细节缺失 | 产品被提及但无功能/参数/客户反馈 | vx100 只列名，无具体功能说明 |
| 行业关键指标缺失 | 根据行业匹配必知指标，检查资料是否涉及 | SaaS 公司没有 NRR/CAC/LTV 数据 |
| 客户集中度风险 | 客户数少或完全缺失 | 仅 2 个客户，前五大占比不明 |
| 竞争格局模糊 | 提竞争优势但没指明对手 | "技术领先"但不知比谁领先 |
| 退出路径未讨论 | PE 投资必须有退出路径 | 无 IPO/并购/回购讨论 |
| 创始人信息不足 | 一级市场核心是投人 | 无创始人履历、过往经历 |

### 行业关键指标速查

| 行业 | 必知指标 |
|------|---------|
| 半导体 | 制程节点、良率、产能利用率、国产替代率 |
| SaaS | NRR/净收入留存率、CAC/LTV、ARR、Churn/流失率 |
| 医疗 | 临床进度、审批状态、医保覆盖、试验数据 |
| 新能源 | 转换效率、度电成本、产能/出货量、补贴依赖度 |
| 消费 | 复购率、渠道结构、客单价、同店增长 |
| AI | 模型性能/精度、推理成本、数据壁垒、API调用量/DAU |
| 制造 | 产能利用率、良率、大客户占比、自动化率 |
| 金融科技 | 风控指标/不良率、牌照、资金成本、交易规模 |
| 通用 | TAM/SAM/SOM、市占率、增速/增长率 |

## 投资问题生成原则

### 问题分类

| 类别 | 关注点 |
|------|-------|
| 商业模式 | 模式是否成立、是否可复制、收入结构是否健康 |
| 收入质量 | 收入是否真实、可持续、是否过度依赖单一来源 |
| 竞争壁垒 | 壁垒是否真实、可持续、能否抵御巨头进入 |
| 团队能力 | 创始人背景、团队完整性、关键人风险 |
| 财务健康 | 现金流、烧钱率、跑道、财务规范性 |
| 估值 | 估值依据、可比公司、增长假设是否合理 |
| 退出 | IPO可行性、并购可能性、回购条款 |
| 风险 | 核心风险是否可控、最坏情况是否可承受 |
| 投资盲区 | 资料未涉及但投决必知 |

### 问题配比

- **60% 基于资料的问题**：矛盾验证、异常追问、弱证据质疑
- **40% 投资盲区问题**：资料未涉及但投决必知

### 投资影响评估

| 影响 | 含义 |
|------|------|
| 正面 | 支持投资 |
| 中性 | 不影响判断 |
| 负面 | 需要关注但可管理 |
| 重大风险 | 可能一票否决 |

## 尽调流程参考

### 标准 PE 尽调清单

1. **业务 DD**：客户验证、供应链核查、产品测试
2. **财务 DD**：审计报告、税务核查、关联交易
3. **法律 DD**：股权结构、知识产权、诉讼风险、合规
4. **团队 DD**：创始人背景调查、关键人依赖、竞业限制

### 从研究到投决的闭环

```
资料加载 → 结构化提取 → 盲区识别 → 问题生成 → 问题回答 
→ 交叉验证 → 论点追踪 → 报告生成 → 投委会材料准备
```

### 报告模式选择

| 模式 | 适用场景 |
|------|---------|
| workbench | 日常推进、项目讨论、快速决策 |
| long | 正式投委会、存档、深度研究 |
