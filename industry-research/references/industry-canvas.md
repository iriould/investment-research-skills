# 行业画布模板

将行业研究的核心发现浓缩在一页画布中，便于快速决策。

## 行业画布

| 区域 | 内容 |
|------|------|
| **行业定义** | 行业名称、边界、细分领域 |
| **市场规模** | TAM / SAM / SOM、CAGR |
| **竞争格局** | 集中度、梯队、头部玩家 |
| **核心技术** | 技术栈层次、架构模式、关键挑战、成熟度、开源vs专有 |
| **价值链** | 上游→中游→下游、利润分布、技术控制点 |
| **政策环境** | 准入门槛、政策趋势、技术监管 |
| **技术趋势** | 当前格局、新兴趋势、技术拐点 |
| **投资 thesis** | 核心逻辑、技术投资逻辑、关键风险、机会窗口 |

## 使用方式

1. 完成行业研究 8 步工作流后，将核心发现填入画布
2. 画布作为投委会讨论的快速参考
3. 后续 skill（公司研究、关键问题、竞对分析）可直接引用画布中的数据
